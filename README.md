# 🛡️ Guardião Digital (Bem-te-vi Digital)

Uma plataforma educacional interativa e gamificada focada em ensinar os fundamentos da **Segurança Digital** para usuários de todas as idades. O sistema utiliza o carismático mascote "Bem-te-vi Digital" para guiar os alunos por uma trilha de aprendizado envolvente.

🌐 **Acesse o projeto online:** [http://bemtevidigitalprotego.com/](https://rafael1102.github.io/Bem-te-vi-digital/)

## 🎯 O que este sistema faz?

O **Guardião Digital** tem como objetivo conscientizar e educar as pessoas sobre as melhores práticas de navegação segura, prevenção de golpes e cidadania digital. Através de uma interface amigável e gamificada, o sistema oferece:

- **Trilha de Aprendizado:** Um mapa de módulos progressivos onde o usuário avança conforme conclui as atividades.
- **Módulos Educacionais:** Cobertura de temas cruciais da atualidade:
  - Fundamentos de Segurança Digital
  - Senhas Fortes e Autenticação (2FA)
  - Redes Seguras e Uso de VPN
  - Prevenção de Golpes Virtuais (Phishing, Engenharia Social)
  - Netiqueta e Convivência Digital (Prevenção ao Cyberbullying)
- **Quizzes e Atividades Interativas:** Avaliações no final de cada unidade com feedback em tempo real para consolidar o conhecimento.
- **Sistema de Progressão (Gamificação):** O usuário ganha pontos de experiência (XP) ao acertar questões e conclui módulos para desbloquear as próximas fases.
- **Perfil Customizável:** Os usuários podem definir seu nome, gênero e acompanhar suas estatísticas (XP, unidades completadas).

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido focado na leveza e acessibilidade, utilizando tecnologias web padrão sem a necessidade de frameworks pesados:

- **HTML5:** Estruturação semântica das páginas.
- **CSS3 (Vanilla):** Estilização completa, layouts responsivos (Flexbox/Grid), temas visuais e animações ricas criadas do zero.
- **JavaScript (Vanilla):** Lógica de negócios, manipulação do DOM, controle de estado via `localStorage` e sistema de validação de quizzes.

## ⚙️ Como executar o projeto

Por ser uma aplicação Frontend puramente baseada em HTML, CSS e JS (Vanilla) e com armazenamento de progresso no navegador (Local Storage), a execução é extremamente simples:

1. Faça o clone ou baixe este repositório.
2. Extraia os arquivos em uma pasta na sua máquina.
3. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão (recomendado: Google Chrome, Firefox ou Edge).
4. Crie sua conta (os dados ficam salvos apenas localmente no seu navegador) e comece a jogar!
